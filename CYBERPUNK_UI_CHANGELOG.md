# NeuroAgent Cyberpunk UI v2.0

## 设计特色

### 🎨 视觉风格
- **深色主题**: 纯黑背景 (#050505) 配霓虹色彩
- **赛博朋克配色**:
  - Cyan (青色): `#00f5ff` - 主要交互色
  - Pink (粉色): `#ff00ff` - 强调色
  - Purple (紫色): `#bf00ff` - 次要色
  - Green (绿色): `#00ff9f` - 成功状态
- **发光效果**: 霓虹光晕、阴影、边框发光
- **Glitch 效果**: Logo 上的故障艺术效果

### ✨ 动画效果
- **动态网格背景**: 透视网格动画，营造深度感
- **浮动光球**: 模糊的渐变光球在背景中浮动
- **扫描线**: 卡片上的水平扫描线动画
- **悬停效果**: 按钮和卡片的发光、位移、光扫效果
- **消息滑入**: 聊天消息的弹性进入动画
- **加载动画**: 多层旋转圆环 + 脉冲大脑图标

### 🖥️ 界面组件

#### 侧边栏
- Syncopate 字体 Logo 带 Glitch 效果
- 激活项的霓虹左边框
- 系统状态面板 (内存使用率、在线指示器)
- 玻璃拟态效果

#### 聊天界面
- 三种记忆模式卡片 (STANDARD/DEEP/QUICK)
- 霓虹边框输入框
- 渐变发送按钮
- 会话历史侧边栏
- 思考过程折叠面板

#### 知识图谱
- D3.js 力导向图
- 节点发光滤镜效果
- 三级节点颜色区分 (Domain粉红/Concept青色/Instance绿色)
- 节点详情面板
- 缩放/搜索/筛选功能

#### 文件上传
- 旋转圆环装饰的上传区域
- 拖放支持
- 文件格式图标网格
- 进度条动画

#### 配置页面
- 滑动开关组件
- 范围滑块
- 统计卡片网格
- 危险操作区域

### 📝 字体设计
- **Space Grotesk**: 主要UI字体，现代几何风格
- **Syncopate**: Logo和标题，科技感
- **JetBrains Mono**: 代码和数值，等宽字体

### 🎯 交互细节
- 所有可交互元素都有悬停效果
- Toast 通知带滑入动画
- 按钮光扫效果
- 卡片扫描线动画
- 滚动条霓虹渐变

## 技术实现

### 使用的库
- Tailwind CSS (CDN)
- Font Awesome 6
- D3.js v7 (仅KG页面)
- Google Fonts

### 自定义 CSS
- CSS Grid 背景动画
- SVG 发光滤镜
- Glitch 动画关键帧
- 玻璃拟态效果
- 霓虹边框和阴影

## 页面路由

| 路由 | 页面 | 特色功能 |
|------|------|----------|
| `/query` | Neural Chat | 记忆模式选择、思考过程展示 |
| `/kg-viz` | Knowledge Graph | D3可视化、节点详情、多层筛选 |
| `/upload` | Data Upload | 拖放上传、格式展示、进度条 |
| `/config` | System Config | API设置、内存参数、存储管理 |

## 启动方式

```bash
python launch_ui.py
```

访问: http://localhost:5000
