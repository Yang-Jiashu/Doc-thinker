# NeuroAgent 简洁 UI

## 特点
- **浅色主题**: 简约清新的设计
- **上传进度条**: 清晰可见的蓝色进度条
- **可用功能**: 上传和对话都能正常工作

## 启动
```bash
python launch_ui.py
```

访问: http://localhost:5000

## 主要功能

### 1. 智能对话 (`/query`)
- 输入问题，按 Enter 发送
- 三种模式：标准/深度/快速
- 新对话按钮清空历史

### 2. 文档上传 (`/upload`)
- 点击或拖拽上传文件
- **显示上传进度条** (0% -> 100%)
- 显示文件状态：等待中 → 上传中 → 完成/失败
- 支持多种格式：PDF, Word, PPT, Excel, 图片, 文本
