# 将UI改造为GPT聊天风格

## 目标
将当前的查询页面改造为类似GPT聊天的界面，实现连续对话历史、消息气泡样式和简洁的用户体验。

## 设计思路
1. **聊天式布局**：采用垂直流式布局，消息按时间顺序排列
2. **消息气泡设计**：区分用户输入（右对齐蓝色气泡）和系统回复（左对齐灰色气泡）
3. **固定输入区域**：将输入框固定在页面底部，方便连续输入
4. **对话历史**：支持查看和管理历史对话
5. **现代简洁风格**：使用更简洁的设计，突出聊天内容
6. **响应式设计**：适配不同屏幕尺寸

## 实现步骤

### 1. 修改 base.html
- 调整CSS样式，为聊天界面添加基础样式
- 优化导航栏，使其更简洁

### 2. 重写 query.html
- 移除现有的左右分栏布局
- 实现聊天容器，支持消息滚动
- 添加消息气泡样式
- 实现固定在底部的输入区域
- 添加对话历史管理功能
- 优化查询模式选择方式

### 3. 增强 JavaScript 功能
- 实现消息发送和接收逻辑
- 添加对话历史存储和管理
- 实现实时打字效果或加载状态
- 添加消息复制、清空等功能

### 4. 保持现有功能
- 保留所有现有查询模式
- 保留结果置信度和来源展示
- 保留查询示例功能

## 预期效果
- 聊天式界面，类似GPT的用户体验
- 支持连续对话
- 清晰的消息区分（用户/系统）
- 简洁现代的设计风格
- 流畅的交互体验

## 文件修改清单
- `raganything/ui/templates/base.html`：添加聊天样式
- `raganything/ui/templates/query.html`：重写为聊天界面

## 技术实现
- 使用现有的Bootstrap 5框架
- 添加自定义CSS实现聊天样式
- 增强JavaScript实现聊天功能
- 保持与现有后端API的兼容性
