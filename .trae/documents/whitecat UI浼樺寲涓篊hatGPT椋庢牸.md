# whitecat UI优化为ChatGPT风格

## 1. 颜色方案调整
- **主色调**：从橙色（#ff6b35）改为ChatGPT蓝色系（#10a37f 调整为 #0052cc）
- **背景色**：优化为更舒适的深色渐变
- **文本颜色**：调整对比度，提高可读性
- **消息气泡**：
  - 用户消息：ChatGPT蓝色（#0052cc）
  - 系统消息：浅灰色（#444654 调整为 #343541 背景下的浅灰色）

## 2. 聊天界面优化
- **消息气泡样式**：调整圆角、内边距和阴影，更接近ChatGPT
- **消息间距**：增加消息之间的垂直间距
- **字体优化**：调整字体大小、行高和字重
- **输入区域**：简化设计，移除不必要的装饰

## 3. 整体布局改进
- **侧边栏**：优化宽度和样式，使其更紧凑
- **聊天区域**：调整最大宽度，确保在大屏幕上有良好的阅读体验
- **滚动条**：进一步优化滚动条样式，使其更隐蔽

## 4. 交互体验增强
- **消息动画**：优化消息出现的动画效果
- **加载状态**：改进打字指示器的样式
- **按钮反馈**：增强按钮的悬停和点击效果

## 5. 响应式设计优化
- 确保在各种屏幕尺寸上都有良好的表现
- 优化移动端布局

## 6. 细节优化
- 调整图标样式和大小
- 优化按钮和输入框的边框样式
- 改进文件上传组件的外观

## 文件修改计划
1. **base.html**：更新整体样式和布局
2. **query.html**：优化聊天界面和输入区域
3. **upload.html**：调整上传页面样式，保持一致性

## 预期效果
- 更接近ChatGPT的现代化对话UI
- 更舒适的配色方案
- 更好的交互体验
- 响应式设计，适配各种设备
