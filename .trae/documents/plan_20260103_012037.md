# 在聊天输入区域添加文件上传功能

## 目标
在聊天界面的textarea左侧添加文件上传接口，允许用户方便地上传文件。

## 设计思路
1. 在textarea左侧添加文件上传按钮
2. 实现文件选择和上传功能
3. 保持与现有设计风格的一致性
4. 支持多种文件格式
5. 显示上传进度和状态

## 实现步骤

### 1. 修改聊天输入表单
- 在textarea左侧添加文件上传按钮
- 使用隐藏的file input元素
- 添加文件上传的CSS样式

### 2. 实现文件上传功能
- 添加文件选择和上传的JavaScript逻辑
- 支持拖拽上传到输入区域
- 显示已选择文件列表
- 支持取消上传

### 3. 集成到现有聊天系统
- 将上传的文件信息发送到服务器
- 在聊天历史中显示文件上传记录
- 支持与文本消息一起发送文件

### 4. 保持设计一致性
- 使用现有的深色主题
- 按钮样式与现有设计匹配
- 上传状态显示样式统一

## 预期效果
- 在textarea左侧显示文件上传按钮
- 支持点击选择文件
- 支持拖拽上传
- 显示上传进度
- 与现有聊天功能无缝集成

## 文件修改清单
- `raganything/ui/templates/query.html`：修改聊天输入区域，添加文件上传功能

## 技术实现
- 使用HTML5 file input
- 添加JavaScript文件上传逻辑
- 保持与现有样式的一致性
- 支持多文件上传
- 显示上传状态和进度
