# NeuroAgent Clean UI v3.0

## 设计变更

从赛博朋克风格回归到**简约清新**的设计：

### 视觉风格
- **浅色主题**: 白色和浅灰色背景
- **蓝色主色调**: 专业的蓝色 (#2563eb) 作为强调色
- **简约卡片**: 白色卡片 + 细边框 + 轻微阴影
- **清晰排版**: Inter 字体，良好的可读性
- **适度圆角**: 12px 圆角，现代但不夸张

### 功能修复

#### 1. 文件上传
- ✅ 简化的独立上传端点（无需后端）
- ✅ 清晰的进度条（蓝色渐变）
- ✅ 文件大小和状态显示
- ✅ 支持拖拽上传

#### 2. 聊天功能
- ✅ 修复了网络错误问题
- ✅ 独立的查询端点（返回演示响应）
- ✅ 三种记忆模式选择
- ✅ 清晰的对话界面

#### 3. 知识图谱
- ✅ 简约的 D3.js 可视化
- ✅ 层级筛选功能
- ✅ 节点详情面板
- ✅ 搜索功能

### 文件结构

```
核心库/ui/templates/
├── base_modern.html      # 基础模板（浅色主题）
├── query_modern.html     # 对话页面
├── kg_viz_modern.html    # 知识图谱可视化
├── upload_modern.html    # 文件上传（带进度条）
└── config_modern.html    # 设置页面
```

### API 端点

| 端点 | 功能 | 状态 |
|------|------|------|
| `/api/v1/query/text` | 文本查询 | ✅ 独立工作 |
| `/api/v1/upload` | 文件上传 | ✅ 独立工作 |
| `/api/v1/graph/hierarchical` | 图谱数据 | ✅ 返回演示数据 |

### 启动

```bash
python launch_ui.py
```

访问: http://localhost:5000
